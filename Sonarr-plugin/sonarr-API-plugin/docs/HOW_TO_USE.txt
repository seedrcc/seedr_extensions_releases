================================================================================
  HOW TO USE SONARRSEEDR
================================================================================

WHAT YOU'LL SEE:
----------------
When you run SonarrSeedr.exe:
1. NO BLACK CONSOLE WINDOW (it runs hidden in background)
2. Browser opens automatically to http://localhost:8242
3. Seedr icon appears in SYSTEM TRAY (near the clock)


HOW TO START:
-------------
Double-click: SonarrSeedr.exe


HOW TO KNOW IT'S RUNNING:
--------------------------
1. Check SYSTEM TRAY (bottom-right, near clock)
   - Look for your Seedr logo icon
   - If not visible, click the UP ARROW (^) to show hidden icons

2. Open browser to: http://localhost:8242


HOW TO CONTROL THE APP:
------------------------
RIGHT-CLICK the tray icon for menu:
  - Open Web Interface (or just double-click icon)
  - Dashboard
  - API Documentation
  - Quit SonarrSeedr


HOW TO STOP:
------------
Method 1: Right-click tray icon → "Quit SonarrSeedr"
Method 2: Task Manager → End "SonarrSeedr.exe" process


TROUBLESHOOTING:
----------------

Problem: Nothing happens when I run it
Solution:
  1. Check Task Manager → Is "SonarrSeedr.exe" running?
  2. Check system tray (click ^ arrow near clock)
  3. Try visiting: http://localhost:8242 in browser

Problem: Can't find the tray icon
Solution:
  1. Click the UP ARROW (^) on taskbar (near clock)
  2. Look for Seedr logo in hidden icons
  3. Drag it to taskbar to keep visible

Problem: Browser doesn't open
Solution:
  1. Manually open browser
  2. Go to: http://localhost:8242
  3. App is still running even if browser doesn't auto-open

Problem: Port 8242 already in use
Solution:
  1. Stop other app using that port
  2. Or run: SonarrSeedr.exe --port 9999 (use different port)

Problem: Want to see error messages
Solution:
  1. Run: debug.bat (shows console window)
  2. Or check: folder_watcher.log


QUICK TIPS:
-----------
✓ App runs in background - NO console window
✓ Browser opens automatically
✓ System tray icon shows it's running
✓ Right-click icon for quick access
✓ Access at: http://localhost:8242
