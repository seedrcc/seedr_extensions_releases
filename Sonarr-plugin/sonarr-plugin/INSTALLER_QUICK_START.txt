=====================================
WINDOWS INSTALLER - QUICK START
=====================================

WHAT THIS DOES:
Makes your app appear in:
  ✓ Windows Settings > Apps & features
  ✓ Control Panel > Programs and Features
  ✓ Start Menu
  ✓ Provides proper uninstaller

STEPS TO CREATE INSTALLER:

1. INSTALL INNO SETUP
   Download: https://jrsoftware.org/isdl.php
   Install to default location (C:\Program Files (x86)\Inno Setup 6\)

2. CUSTOMIZE YOUR INSTALLER
   Edit: installer.iss
   
   Change these lines:
   - Line 6: #define MyAppPublisher "Your Name"
   - Line 7: #define MyAppURL "https://yourgithub.com/..."
   - Line 9: #define MyAppId (generate new GUID at guidgenerator.com)

3. BUILD THE INSTALLER
   Double-click: build-installer.bat
   
   Wait for it to complete. Output will be in:
   releases\installers\SonarrSeedr-Setup-v1.1.0.exe

4. TEST IT
   - Run the installer (as Administrator if needed)
   - Install the application
   - Open Windows Settings > Apps
   - Search for "Sonarr-Seedr Integration"
   - You should see it listed!

5. DISTRIBUTE
   Give users the installer .exe file instead of the ZIP file
   
   They install it, and it shows up in their Windows app list!

=====================================

IMPORTANT NOTES:

- Version is auto-read from app\version.py
- Installer creates uninstaller automatically
- Users can uninstall from Windows Settings
- Creates Start Menu and Desktop shortcuts (optional)

=====================================

WHAT IF I WANT BOTH?

You can offer BOTH options:

1. INSTALLER (.exe) - For most users
   - Professional installation
   - Shows in Windows apps
   - Easy uninstall

2. PORTABLE (.zip) - For advanced users  
   - No installation needed
   - Run from anywhere
   - No Windows registration

Just provide both files in your releases!

=====================================

TROUBLESHOOTING:

"Inno Setup not found"
  → Install Inno Setup 6 first
  → Restart terminal

"Build failed"
  → Make sure dist\SonarrSeedr\SonarrSeedr.exe exists
  → Run build.bat first if needed

"App doesn't show in Windows list"
  → Must use INSTALLER, not ZIP
  → Generate unique AppId (GUID)
  → Install as Administrator
