## ⚡ Super Quick "Just Run It"

1. **Double-click** `SonarrSeedr.exe`
2. **Open** http://localhost:8000
3. **Scan QR code** with your phone to authenticate
4. **Done!** (Well, mostly - you'll need Sonarr/Prowlarr configured)

## ✅ Prerequisites Checklist

Before you start, make sure you have:

- [ ] **Sonarr** installed and running on http://localhost:8989
  - Download: https://sonarr.tv/#download

- [ ] **Prowlarr** installed and running on http://localhost:9696
  - Download: https://prowlarr.com/#downloads

- [ ] **Seedr account** (free works!)
  - Sign up: https://seedr.cc

**Don't have them yet?** No problem! Get them first, then come back.
