HOW IT WORKS
=====================================

The app now automatically detects its mode:

1. PORTABLE (from ZIP)
   - No .installed marker file
   - Stores data NEXT TO EXE
   - Truly portable (USB drive ready)
   
2. INSTALLED (via installer)
   - Has .installed marker file
   - Stores data in AppData
   - No permission issues

=====================================

Portable ZIP:
  build.bat
  → releases\SonarrSeedr-v1.1.9-[timestamp].zip

Installer:

  → releases\installers\SonarrSeedr-Setup-v1.1.9.exe
